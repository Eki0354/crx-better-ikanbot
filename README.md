# CRX-Better-Ikanbot

Ikanbot的播放器操作起来实在有些呆，于是用插件替换了一个更聪明的。

## WXT + Vue 3

使用 WXT + Vue 3框架，播放器选择的是[Plyr](https://plyr.io/)。

## 已知问题

+ 主打一个能跑就行，所以页面中切换剧集无效，需要手动再刷新一下页面。

+ 没有任何报错拦截，健壮性堪忧。
